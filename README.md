# Ecommerce Backend API

## Features
- Product APIs
- Cart APIs
- Order APIs
- JWT dependency included
- MySQL integration
- Gradle build system

## Run Project
```bash
./gradlew bootRun
```

## Database
Create MySQL database:

```sql
CREATE DATABASE ecommerce_db;
```

## API Endpoints
- GET /products
- POST /products
- GET /cart
- POST /cart
- GET /orders
- POST /orders
